# Commande `/st` - Script Test jusqu'à réussite

Quand l'utilisateur écrit `/st`, exécuter les scripts de test jusqu'à ce qu'ils réussissent.

Scripts à tester dans l'ordre:
1. `scripts/test-batch-optimization.js`
2. Autres scripts de test si nécessaire

Continuer jusqu'à ce que tous les tests retournent un code de sortie 0.
