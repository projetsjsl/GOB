Instructions de Debogage

Si vous ne voyez pas les modifications :
  1. Vider le cache : Ctrl+Shift+R (Windows) ou Cmd+Shift+R (Mac)
  2. Navigation privee : Ouvrir une fenetre privee et tester
  3. Verifier l'URL : https://gobapps.com/3p1/dist/index.html
  4. Console navigateur : F12 pour voir les erreurs
